Tipos de algoritmos

Existem vários tipos de algoritmos, alguns dos quais são classificados de acordo com seu design, finalidade ou método de resolução de problemas.

  • Algoritimo sequenciaisão são instruções executadas em sequência, uma após a outra, sem ramificações ou repetições. Eles seguem um fluxo linear para resolver um problema.

    • Algoritmos Sequencias

  • São instruções que são executadas em sequência, uma entre outras, sem ramificações e repetições. Siga um fluxo linear para resolver um problema.

    • Algoritimo Condicionais

  • Esses algoritmos implicam tomar decisões com base em condições específicas. Utilize estruturas como "si-entonces" (if-else) para realizar diferentes ações dependendo do cumprimento de certas condições.

    • Algoritimos Recursivos

  • Se dividem em subproblemas mais pequenos do mesmo tipo que o problema original, resolvendo cada subproblema até chegar à solução do problema principal. A recursividade implica uma função que se chama a mesma coisa.
